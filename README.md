# simple-seeder

Dead simple Hypercore seeder

```
npm install -g simple-seeder
```

## Usage

```
simple-seeder -c <hypercore key> -c <hypercore key 2>
```

## File
Alternatively, a file containing the seeds list:

```
simple-seeder --file ./seeds.txt
```

`seeds.txt`
```
core <hypercore key>
bee <hyperbee key>
drive <hyperdrive key>
seeder <seeder key>
```
