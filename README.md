# simple-seeder

Dead simple Hypercore seeder

```
npm install -g simple-seeder
```

## Usage

```
simple-seeder -k <hypercore key> -k <hypercore key 2>
```

## File
Alternatively, a file containing the seeds list:

```
simple-seeder --file ./seeds.txt
```

`seeds.txt`
```
key <hypercore key>
key <hypercore key 2>
bundle <bundle key>
seeder <seeder key>
```
